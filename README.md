## Deskripsi Project
Aplikasi ini adalah sistem kasir (POS) untuk toko sembako yang digunakan untuk mengelola produk dan transaksi penjualan.

---

## Fitur
- Login & Logout (Auth System)
- Manajemen Produk (CRUD)
- Transaksi Penjualan
- Perhitungan otomatis (total, bayar, kembalian)
- Role admin & kasir

---

## Database
- users
- produk
- transaksi
- detail_transaksis

---

## Sistem Login
Menggunakan Laravel Authentication dengan password terenkripsi (bcrypt).

---

## Teknologi
- Laravel
- Blade Template
- MySQL
- Tailwind CSS
